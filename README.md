# Zentara Homepage (Next.js)

## Setup

1. Install dependencies

```bash
npm install
```

2. Configure EmailJS

Create `.env.local` in the project root (you can copy `.env.local.example`):

```bash
cp .env.local.example .env.local
```

Fill in your EmailJS values.

3. Run

```bash
npm run dev
```

Open `http://localhost:3000`.

## Pages

- `/` Home
- `/offer` Offer / contact form
